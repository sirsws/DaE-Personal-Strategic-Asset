# DaE 研究仓

<p align="center">
  <a href="./README.md">English</a> | <a href="./README_CN.md">简体中文</a>
</p>

DaE 仍然是 **主动知识建模（AKM）** 在 Persona / 顾问协作场景中的首个完整分支实现所对应的原始研究线。

这个仓库现在应被理解为 DaE 论文线的**历史公开研究归档层**；当前的方法框架入口已经转入 AKM 母港。

## 快速入口

- **AKM 母港：** https://github.com/sirsws/akm
- **AKM 中的 DaE 分支：** https://github.com/sirsws/akm/tree/main/branches/dae
- **AKM 中的 DaE 论文入口：** https://github.com/sirsws/akm/tree/main/branches/dae/paper
- **AKM 母论文入口：** https://github.com/sirsws/akm/tree/main/papers/akm

## 这个仓库保留什么

- DaE 论文线的原始公开 PDF 资产
- 面向研究的源材料与 prompt
- AKM 结构成型前的公开论文历史参照

## 原始公开论文版本

- **arXiv 风格稿：** *Dialogue as Elicitation: Building High-Fidelity Personas for Personalized AI Advisors with LLMs*
- **SSRN 版本：** *Reducing Alignment Debt in AI Advisory: The Dialogue-as-Elicitation Approach*

## 仓内公开文件

- `./papers/dae_ssrn.pdf`
- `./papers/dae_arxiv_draft.pdf`
- `./src/dae_minimal_prompt.txt`

## 在 AKM 结构中的角色

AKM 现在负责方法论母框架。
这个仓库负责保留 DaE 的历史研究线，并维持原始论文资产的公开可达性。

## 一句话概括

这个仓库保存 DaE 的论文历史；AKM 负责当前的方法框架入口。
